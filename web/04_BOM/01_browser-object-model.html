브라우저 제어 (Browser Object Model)

    `window` 객체를 통해 자바스크립트는 HTML 문서(`document`)를 넘어
    브라우저 창 자체를 제어하고, 사용자와 직접 대화할 수 있습니다.
    `window`는 전역 객체이므로 생략 가능합니다. (예: `alert()`는 사실 `window.alert()`)

1. 사용자와 대화하기: `alert`, `confirm`, `prompt`

    사용자에게 간단한 메시지를 보여주거나, 질문하고, 입력을 받는 기본적인 대화 상자입니다.

2. 페이지 제어하기: `location` 객체 (가장 중요!)

    `location` 객체는 현재 페이지의 주소(URL) 정보를 담고 있으며,
    다른 페이지로 이동(redirect) 시키거나 새로고침할 때 사용됩니다.

3. 방문 기록 제어하기: `history` 객체

    `history` 객체로 브라우저의 '뒤로가기', '앞으로가기' 버튼 기능을 흉내 낼 수 있습니다.

4. 브라우저/화면 정보 얻기 (참고용)

    `navigator`와 `screen` 객체를 통해 사용자의 브라우저 종류, 운영체제, 화면 크기 등의
    정보를 얻을 수 있습니다. (주로 통계 수집이나 반응형 처리의 보조 수단으로 사용)
